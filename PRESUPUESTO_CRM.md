# Presupuesto de Desarrollo: CRM IDG

### Desglose de Servicios Técnicos y Hitos Realizados

| Concepto / Tarea Principal | Descripción Técnica del Trabajo Realizado | Valor de Mercado |
| :--- | :--- | :--- |
| **1. Arquitectura Base y Refactorización** | Conversión del diseño HTML monolítico a un sistema modular moderno (React 19 + Vite). Creación del sistema de enrutamiento y rediseño de UI responsiva (Móvil/PC). | 450 € |
| **2. Backend: Base de Datos en Tiempo Real** | Eliminación de datos falsos (local) e integración del SDK de *Firebase Firestore*. Creación del servicio CRUD para leer, crear, modificar y borrar datos desde la nube. | 600 € |
| **3. Sistema de Usuarios y Seguridad** | Implementación de *Firebase Auth* con Login cifrado. Creación del sistema de "Permisos por Roles" y blindaje de la base de datos con reglas `firestore.rules`. | 400 € |
| **4. Adaptación y Desarrollo de Módulos Core** | Programación e integración de datos en: Clientes, Obras (con Gantt visual), Presupuestos, Facturación, Trabajadores, RRHH y Configuración. | 550 € |
| **5. Creación de Nuevos Módulos** | Ingeniería desde cero de los nuevos módulos solicitados: **Pedidos** (asociado a obras) y **Proveedores** (asociado a materiales y logística). | 350 € |
| **6. Integraciones API y UX** | Sistema dinámico de mensajería API por WhatsApp. Rejillas de tablas con scroll táctil en móviles y menú lateral desplegable (*Off-canvas*). | 150 € |
| **7. Infraestructura, Pruebas y Despliegue** | Configuración de emuladores locales para testeo de datos y despliegue final en dominio de producción (*Firebase Hosting, certificados HTTPS*). | 200 € |
